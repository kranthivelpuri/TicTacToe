<h1>Tic Tac Toe</h1>

A simple <b>Tic Tac Toe</b> game built in <b>Python<b/> using the <b>Tkinter</b> library. This project features an interactive graphical user interface (GUI) that allows two players to play the game with automatic win detection, draw handling, and turn switching.  

<h2>Features</h2>  
- Two-player mode with alternating turns.  
- Automated win detection for rows, columns, and diagonals.  
- Draw condition handling.  
- User-friendly GUI created with Tkinter.  

<h2>How to Run </h2>
1. Clone the repository:    
    git clone https://github.com/yourusername/tic-tac-toe.git  
   
2. Navigate to the project folder:  
    cd tic-tac-toe    
3. Run the Python script:  
    python tictactoe.py
   
   Make sure you have Python 3.x installed.  

<h2>Gameplay</h2>
- Click on a square to make a move.  
- The current player's symbol (`X` or `O`) will appear on the button.  
- The game announces the winner or a draw through a popup message.  
- To play again, restart the application.  

<h2>Project Structure </h2> 
- `tictactoe.py`: The main script containing the game logic and GUI implementation.  

<h2>Requirements  </h2>
- Python 3.x  
- Tkinter (comes pre-installed with Python for most distributions).
